Time Series Prediction -- Signal Processing
===========================================


(A) Sunspots Data:

 - To plot the AR(15) predictions : ``python time_series.py 0``
 - To plot sunspots AR(p) models' error function : `` python time_series.py 1``
 - To plot sunspots prediction using neural nets : `` python time_series.py 2``
 - To plot sunspots Neural-Net error function (this might take a long time): ``python time_series.py 3``


(B) CO2 Data:

 - To predict CO2 data using model decomposition: ``python time_series.py 4``
 - To predicit CO2 data using gaussian process regression: ``python time_series.py 5``


(C) Financial Data:

- To plot the error-rates of the neural-nets :  ``python finance.py 1``
- To train a neural network :   ``python finance.py 0``
